# Introduction to vroomrs

vroomrs is Sentry's high-performance profiling library - a Python extension module written in Rust that processes and derives data from your application profiles. Built for speed and efficiency, it provides powerful tools for analyzing performance data, extracting function metrics, and detecting performance issues.

## What is vroomrs?

vroomrs is a Python extension module that combines Python's ease of use with Rust's performance. It processes profiling data to help you understand your application's performance characteristics, identify bottlenecks, and detect performance issues automatically.

The library supports both single profiles and continuous profiling through profile chunks, making it suitable for various profiling scenarios from one-time analysis to production monitoring.

## Why use vroomrs?

### Performance at scale

Processing profiling data can be computationally expensive. vroomrs leverages Rust's performance to handle large profiles efficiently, making it suitable for production environments where you need to process thousands of profiles.

### Automatic issue detection

Rather than manually analyzing profiles, vroomrs can automatically detect common performance issues:

- Frame drops caused by main thread blocking
- Slow operations on the main thread (I/O, compression, database)
- SwiftUI performance issues (view inflation, layout, rendering)
- Machine learning model operations
- Platform-specific performance patterns

### Flexible analysis

Whether you're analyzing a single profile or processing continuous profiling data, vroomrs provides the tools you need:

- **Profile**: For transaction-based profiling with a defined start and end
- **ProfileChunk**: For continuous profiling that captures ongoing application behavior

### Easy integration

As a Python extension module, vroomrs integrates seamlessly into your Python workflow while providing the performance benefits of native code.
