## Quick example

This guide will walk you through the basics of using vroomrs to process profiling data, extract function metrics, and detect performance issues.

<Steps>
  <Step title="Import vroomrs">
    Start by importing the vroomrs module:

    ```python
    import vroomrs
    ```
  </Step>

  <Step title="Load a profile">
    Load a profile from a JSON string:

    ```python
    # Load a profile from JSON
    with open("profile.json", "r") as f:
        profile_json = f.read()

    profile = vroomrs.profile_from_json_str(profile_json)
    ```

    <Tip>
    If you know the platform in advance, pass it for better performance:
    ```python
    profile = vroomrs.profile_from_json_str(profile_json, platform="android")
    ```
    </Tip>
  </Step>

  <Step title="Access metadata">
    Extract metadata from the profile:

    ```python
    # Get basic information
    profile_id = profile.get_profile_id()
    platform = profile.get_platform()
    duration = profile.duration_ns()

    print(f"Profile ID: {profile_id}")
    print(f"Platform: {platform}")
    print(f"Duration: {duration / 1_000_000}ms")
    ```
  </Step>

  <Step title="Extract function metrics">
    Analyze function performance:

    ```python
    # Extract function metrics
    functions = profile.extract_functions_metrics(
        min_depth=1,
        filter_system_frames=True,
        max_unique_functions=10
    )

    # Find the slowest functions
    for func in functions:
        print(f"{func.get_function()}: {func.get_sum_self_time_ns() / 1_000_000}ms")
    ```
  </Step>

  <Step title="Detect performance issues">
    Automatically find performance problems:

    ```python
    # Find performance issues
    occurrences = profile.find_occurrences()

    for occurrence in occurrences.get_occurrences():
        print(f"{occurrence.get_type()}: {occurrence.get_issue_title()}")
        print(f"  Culprit: {occurrence.get_culprit()}")
        print(f"  Duration: {occurrence.get_duration_ns() / 1_000_000}ms")
    ```
  </Step>
</Steps>

## Working with compression

vroomrs supports LZ4 compression for efficient storage:

```python
# Compress a profile
compressed = profile.compress()

# Save to file
with open("profile.lz4", "wb") as f:
    f.write(compressed)

# Load compressed profile later
with open("profile.lz4", "rb") as f:
    compressed_data = f.read()

profile = vroomrs.decompress_profile(compressed_data)
```

## Using ProfileChunk

For continuous profiling, use `ProfileChunk`:

```python
# Load a profile chunk
chunk = vroomrs.profile_chunk_from_json_str(chunk_json, platform="android")

# Access chunk-specific metadata
chunk_id = chunk.get_chunk_id()
profiler_id = chunk.get_profiler_id()
duration_ms = chunk.duration_ms()

print(f"Chunk ID: {chunk_id}")
print(f"Duration: {duration_ms}ms")

# Extract metrics from chunks the same way
functions = chunk.extract_functions_metrics(
    min_depth=1,
    filter_system_frames=True
)
```

## Complete example

Here's a complete example that puts it all together:

```python
import vroomrs

# Load and normalize a profile
with open("profile.json", "r") as f:
    profile_json = f.read()

profile = vroomrs.profile_from_json_str(profile_json, platform="android")
profile.normalize()

# Get profile metadata
metadata = profile.get_metadata()
print(f"Platform: {profile.get_platform()}")
print(f"Duration: {profile.duration_ns() / 1_000_000}ms")

# Extract and analyze function metrics
functions = profile.extract_functions_metrics(
    min_depth=1,
    filter_system_frames=True,
    max_unique_functions=20
)

print(f"\nTop {len(functions)} slowest functions:")
for i, func in enumerate(functions, 1):
    self_time_ms = func.get_sum_self_time_ns() / 1_000_000
    sample_count = func.get_sample_count()
    print(f"{i}. {func.get_function()}: {self_time_ms:.2f}ms ({sample_count} samples)")

# Detect performance issues
occurrences = profile.find_occurrences()

if occurrences.get_occurrences():
    print(f"\nFound {len(occurrences.get_occurrences())} performance issues:")
    for occurrence in occurrences.get_occurrences():
        print(f"  - {occurrence.get_type()}: {occurrence.get_issue_title()}")
```
